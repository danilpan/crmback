## Crmka
[Crmka.pro](http://crmka.pro/)


#### Требования к серверу
1. Операционная система: Ubuntu 16.04 или выше
2. Дополнительно на сервере должно быть:
    * nginx
    * php 7.2
    * PostgreSQL 
    * Redis
    * ElasticSearch не ниже 5
    * supervisor
