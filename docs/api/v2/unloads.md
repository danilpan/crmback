#### Получение одной выгрузки по ID:
**GET:** */api/v2/unloads/{id}*  

Параметры:
- **id**: *обязательный*, ID выгрузки. 

Пример ответа:  
```
{
    "id": 6,
    "name": "Выгрузка-6",
    "comment": "Описание выгрузки 6",
    "organization_id": "1",
    "config": "{data:{}}",
    "api_key": "d5a34032698b3c32a52b54d5d5f56e61",
    "is_work": 1
}
```

#### Редактирование выгрузки:
**PUT:** */api/v2/unloads/{id}*

Параметры:
- **id**: *обязательный*, ID выгрузки.

Ограничение по полям:   
- **name**: *обязательный*, Название выгрузки.
- **comment**: *обязательный*, Комментарий выгрузки.
- **organization_id**: *обязательный*, Id-организации.
- **config**: *обязательный*, Поля и значения выгрузки.
- **is_work**: *обязательный*, Статус выгрузки.

Все остальные поля являются необязательными.
   
Пример запроса:
```
{
	"name": "Выгрузка-7",
	"comment": "Описание седьмой выгрузки",
	"organization_id": 1,
	"config": "{data:{}}",
	"is_work": 1
}
```  

Пример ответа:  
```
{
    "id": 7,
    "name": "Выгрузка-7",
    "comment": "Описание седьмой выгрузки",
    "organization_id": 1,
    "config": "{data:{}}",
    "api_key": "01fcefaec564ae6bca70f20c0c1d0044",
    "is_work": 1
}
```

#### Удаление продукта:
**DELETE:** */api/v2/unloads/{id}*

Параметры:
- **id**: *обязательный*, ID выгрузки.
   
Пример запроса:
```
Method: Delete.
Url: http://crmka.lcl/api/v2/unloads/7
```  

Пример ответа:  
```
1
```
