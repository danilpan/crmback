### Организации

#### Получение списка организаций по ID родителя
**GET:** */api/v2/organizations/{id}/children*   

Параметры:
- **id**: *обязательный*, ID родительской организации. 
При указании 0 будут выбраны организации верхнего уровня.

Пример ответа:  
```
{
    "total": 2,
    "data": [
        {
            "id": 1,
            "title": "Компания 1",
            "parent_id": 0
        },
        {
            "id": 2,
            "title": "Компания 2",
            "parent_id": 0
        }
    ]
}
```

#### Получение организации по ID:
**GET:** */api/v2/organizations/{id}*  

Параметры:
- **id**: *обязательный*, ID организации. 

Пример ответа:  
```
{
    "id": 2,
    "title": "Продажи",
    "parent_id": 1,
    "path": [
        {
            "id": 1,
            "title": "renam",
            "parent_id": null
        }
    ]
}
```

#### Редактирование организации:
**PUT:** */api/v2/organizations/{id}*

Параметры:
- **id**: *обязательный*, ID организации.

Ограничение по полям:   
- **title**: *обязательный*, Название организации.
   
Пример запроса:
```
{
    "title" : "Новое название"
}
```  

Пример ответа:  
```
{
    "id"    : 34, 
    "title"     : "Новое название",
    "parent_id" : 19
}
```

#### Создание организации:
**POST:** */api/v2/organizations/{id}/children*

Параметры:
- **id**: *обязательный*, ID родительской организации. 
При указании 0 будет создана организация верхнего уровня.

Ограничение по полям:   
- **title**: *обязательный*, Название организации.
   
Пример запроса:
```
{
    "title" : "Новая компания"
}
```  

Пример ответа:  
```
{
    "id"    : 34, 
    "title"     : "Новая компания",
    "parent_id" : 19
}
```
